What Is SmartOS?

OpenSolaris

  • In 2005 Sun Microsystems open sourced Solaris
  • In 2007 Sun released a full distrobution called OpenSolaris
  • This move was prompted by pressure from Linux as it gained enterprise traction
  • The BSD community was the real winner here:
    • License allowed closed source derivatives => GPL incompatible
    • Sun middle management was too controlling failed to nurture a community
  • Fast forward 2011:
    • Oracle has assimilated Sun
    • OpenSolaris has become Solaris 11 Express and source is closed
    • Many influential kernel developers have defected to startups
    • This is a good thing!

What Is SmartOS?

Illumos

  • Illumos is the core OS forked from the last released build of OpenSolaris
  • Developed lead by those defected kernel developers
  • Sponsored by the startups they work for (Joyent, Nexenta, Delphix etc)
  • Talent behind the core Solaris innovations (ZFS, Zones, DTrace ) are now part of the Illumos community

What is SmartOS?

Joyent

  • Built SmartOS and ported KVM over to Illumos
  • Runs it's own public cloud which competes with Amazon
  • Will license SmartDataCenter (orchestration layer) so you can compete with Amazon as well
  • Are the corporate sponsor of Node.JS

What is SmartOS?

SmartOS ...

  • A minimalist Illumos distrobution designed to be hypervisor
  • Brings the powerful features of Solaris to Linux host management
  • Small footprint
  • Safely and easily remotely updatable
  • Free, open source with an active community

SmartOS Advantages

ZFS

  • Export block devices for VM use
  • Linux sees a simple single drive, in reality a ZFS RAID beast
  • Instaneous incremental live snapshots of block devices
  • Snapshots can be easily exported/imported
  • Near instaneous local cloning of VMs
  • Clones initially take up virtually no additionalspace

SmartOS Advantages

Zones

  • Zones are Operating System level virtualization abstractions
  • Secure
  • Low overhead (CPU, Memmory)
  • Fine grained resource control (CPU, Memmory, IO)
  • KVM can be run inside zones and benefit from their features

SmartOS Advantages

DTrace

  • DTrace provides fast and safe runtime system introspection
  • Lock profiling, Kernel/Process stack traces, System call latencies, etc
  • KVM on SmartOS includes new DTrace Providers for:
    • Virutal Disk Throughput
    • Virtual NIC Throughput
    • Virtual Disk IOps
    • Hardware Interrupts
    • Virtual Machine Exits
    • vCPU Samples

SmartOS Advantages

Node.js

  • Use JavaScript to build fast, scalable network programs easily
  • Joyent is the corporate sponsor and runs http://no.de
  • Joyent ommitted to making Illumos the premier platform for Node.js:
    • DTrace providers for safe runtime introspection
    • MDB (Solaris Debugger) modules for inspecting state, including core dumps
    • Support

Concerns

  • KVM port relatively new though Joyent running it in production
  • SmartOS/SmartMachines are Unix systems, but it's not Linux
  • General Illumos documentation is sparse (Solaris 11 Express documentation used in practice)
  • Smaller Community
